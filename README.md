# Dynamic Web Scraper
> Scrapes a Website that is taken from an inputted emails domain.

This project uses ECMAScript 6(2015)

## Installation

Windows & OS X & Linux:

```sh
git pull https://github.com/Crithane/dynamicWebScraper.git
npm install
```

## Usage example

```sh
node app.js johndoe@example.com
```
